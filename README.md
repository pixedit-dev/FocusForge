## Installation & Setup

1. Clone the repository:

```
git clone https://github.com/pixedit-dev/FocusForge.git
```

2. Navigate to the project folder:

```
cd FocusForge
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```
